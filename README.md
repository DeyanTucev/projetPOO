## **Project Setup & Installation**
### **Install Dependencies**
Run the following command to install all necessary dependencies:  
```bash
npm install
```
Install Axios if not already done:
```bash
npm install axios
```
To run the project:
```bash
npm run serve
```
If any issues arise, use lint:
```bash
npm run lint
```
or prettier:
```bash
npx prettier . --write
```
